HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

'br' change the line, and it doesn't have to be closed.

The other hand, 'p' means paragraph. It needs to be closed.

cons of 'p' is the fact that the gap between two paragraphs are fixed.
However, you don't have to worry about it as there is a computer language to support the 'p' called CSS.

style="margin-top:45px;" following after 'p' makes 45px of extra margin(gap).
